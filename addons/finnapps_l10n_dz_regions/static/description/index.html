Wilayas d’Algérie, localités et codes postaux.

Gestion par base de données relationnelles des adresses de vos contacts localisé en Algérie.

Fonctionnalités principales

  • Organisation rationnelle des adresses des contacts.
  • Ajout des Wilayas Algériennes.
  • Ajout de 1540 communes.
  • Ajout de 16 096 localités et leurs codes postaux.
  • Création d’une liaison entre les Wilayas et leurs localités.

Fonctionnalités

  • Créer une relation dynamique entre les Wilayas d’Algérie et leurs Localités respectives.
  • Renseigne automatiquement le code postal après avoir choisi la localité.


    • Fonctionnement

    Sur la fiche « contact » :
  • Après avoir choisi le pays « Algérie », les wilayas d’Algérie deviennent disponibles à la sélection dans le champ « Wilaya ».
  • Après avoir choisi la wilaya, les localités rattachées à celle-ci deviennent disponibles à la sélection dans le champ « Localité ».
  • Une fois la localité sélectionnée, le code postal correspondant est automatiquement repris dans le champ « Code postal ».


    • Avantages

  • Evite la création de doublant dans la liste des wilayas ou des localités.
  • Permet ainsi une analyse correcte de vos contacts sur la base de leurs localisation (Wilaya / Localités / code postal).

Implémentation

  • Alimente la table « Pays = Algérie », avec les Wilayas Algériennes et leurs codes postales respectives.
  • Alimente la table « Etat fédéraux = Wilayas Algériennes », avec leurs localités et leurs codes postaux respectives.
  • Le nombre de localités ajoutés est de : 16 096.
  • En suivant le chemin Contacts/Configuration on trouvera les menus “Communes” et “Localités” où on peut consulter les communes et les wilayas de l’Algérie.
