 Contrat


 Pour que le salaire soit correctement calculé et généré, il est nécessaire que le contrat soit convenablement rempli (champs obligatoires) et qu’il soit en cours de validité.


 Attribution de congé

 A la création d’un nouveau contrat pour un employé, le système attribue 2.5 jours de congé par mois, en allocations de type “provisionnelle”.

 Il est possible de modifier le nombre de jours de congé attribués par mois.

 Il est aussi possible d'ajouter manuellement des jours de congé, nommés sur odoo “Jours extra”.


 Congé et approbation

 Type de congé                                                   

 Les types suivants ont été traités:                                          


  • Congé Payé.
  • Absence.
  • Absence autorisée.
  • Congé maladie.
  • Jours de compensation.

 Les jours d’absence sont déduits automatiquement du nombre de jours travaillés par mois.

 Approbation                                                   

 Les jours de congé d’un employé doivent être approuvés par son responsable.

 Ce responsable est nommé sur odoo Validateur / Congé et se trouve sur la fiche employé.

 Validation                                                   

 Une fois que l'employé est en congé, le responsable / congé doit valider le congé.


 Règles salariales

 Les règles salariales algériennes ont été implémentées dans les structures salariales.


 Structures Salariales

 En fonction du contrat de l'employé, une des 07 structures est utilisée pour le calcul de la paie.

 La valeur la plus commune en Algérie est “Structure Employée”.


 Bulletin de paie

 Sur le workflow standard de odoo, il est possible de générer les bulletins de paie unitairement ou par lot.

 Le bulletin de paie est calculé en fonction de la structure salariale choisie.

 Toutes les informations obligatoires figurent sur le bulletin tel que :

  • Les informations relatives à la société.
  • Les informations relatives à l'employée.
  • Les informations relatives à la composition du salaire.

 Les jours non travaillés sont déduits automatiquement du calcul du salaire.

 Il est possible de configurer le nombre de jours travaillés par semaine et par employée sur le contrat:

  • Standard Algérie 5 Jours par semaine.
  • Standard Algérie 6 Jours par semaine.
  • Standard Algérie 7 Jours par semaine.

 Cette dernière configuration permet d’avoir automatiquement la base du mois : 22, 26 ou 30 Jours.


 Attestation de travail et de salaire

 Possibilité de produire et d'imprimer une attestation de travail et de salaire.

 Contract

 For the salary to be correctly calculated and generated, it is necessary that the contract is properly completed (required fields) and that it is valid.


 Attribution

 When creating a new contract for an employee, the system allocates 2.5 days of leave per month, in “provisional” type allowances.

 It is possible to modify the number of days of leave allocated per month.

 It is also possible to manually add days off, named on odoo “Extra days”.


 Leave and approval

 Leave types                                                   

 The following types were treated:                                          


  • Paid leave.
  • Sick leave.
  • Absence.
  • Authorized absence.
  • Clearing days.

 Days of absence are automatically deducted from the number of days worked per month.

 Approval                                                   

 An employee's days off must be approved by their manager.

 Ce responsable est nommé sur odoo Validateur / Congé et se trouve sur la fiche employé.

 Validation                                                   

 Once the employee is on leave, the manager / leave must validate the leave.


 Salary rules       

 Algerian salary rules have been implemented in salary structures.


 Salary structures

 Depending on the employee's contract, one of the 07 structures is used for the payroll calculation.

 The most common value in Algeria is “Structure Employée”.


 Pay slip

 On the standard odoo workflow, it is possible to generate pay slips individually or by batch.

 The payslip is calculated according to the chosen salary structure.

 All mandatory information appears on the bulletin such as:

  • Company information.
  • Employee information.
  • Information relating to the composition of the salary.

 Days not worked are automatically deducted from the salary calculation.

 It is possible to configure the number of days worked per week and per employee on the contract:

  • Standard Algeria 5 days per week.
  • Standard Algeria 6 days per week.
  • Standard Algeria 7 days per week.

 This last configuration allows you to automatically have the base of the month: 22, 26 or 30 Days.
