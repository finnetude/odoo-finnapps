État Récapitulatif Annuel E.R.A

Impression de rapport ERA après Calcul relatifs à la comptabilité Algérienne

  • Aperçu
  • Versions

Le calcul des lignes de rapport “État récapitulatif annuel (ERA)” est basé sur une configuration où vous trouveras les comptes comptables qui sont pris on compte sur le calcul de chaque ligne.

Une fois cliquer sur le menu “État récapitulatif annuel (ERA)”, un wizard s'affiche afin de remplir l'année fiscale, puis calculer les lignes de rapport ERA .

Enfin vous aurez le rapport “État récapitulatif annuel (ERA)” en format PDF
