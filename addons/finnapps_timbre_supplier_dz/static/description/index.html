Droit de Timbre sur Facture et sur Paiement Fournisseur

Ce module permettant de gérer les droits de timbre pour les paiements en liquide sur les factures fournisseurs et les avoirs fournisseurs.

  • Fonctionnalités
  • Aperçu
  • Versions

Fonctionnalités

Intégrer le droit de timbre fournisseur sur les factures d’achat ou les paiements fournisseurs.
Pour le paiement intégral d’une facture d’achat, le système prend en compte le montant de la facture fournisseur et celui du droit de timbre.

Configuration

  • Saisir les champs dans la configuration:
    • Compte contrepartie achat: Utilisé pour les écritures comptables du droit de timbre fournisseur.
    • Se baser:
      • Sur validation de Facture : gère le droit de timbre fournisseur sur les factures fournisseur.
      • Sur les Paiements en espèces : gère le droit de timbre fournisseur sur la base des paiements fournisseur.
  • Un journal de droit timbre fournisseur est ajouté par défaut à l'installation du module.

    • Se baser sur la validation des factures

  • Définir sur une facture client un mode de paiement, s'il est de type liquidité, un droit de timbre est visible à remplir.
  • Le montant du droit de timbre est ajouté au montant à payer de la facture fournisseur.

    • Se baser sur paiements des factures

  • Une fois que vous aurez sélectionné le journal "Espèces", le groupe "TIMBRE FOURNISSEUR" sera alors visible.
  • Après avoir rempli le montant du droit de timbre fournisseur, une pièce comptable propre à celui–ci est créée.
  • Il est également possible d'inclure un droit de timbre fournisseur au moment de l'enregistrement des paiements pour les factures fournisseurs.
